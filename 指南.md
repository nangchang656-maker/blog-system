项目名称：智墨博客
项目描述：
智墨博客是一款前后端分离的个人博客平台。旨在提升个人内容创作和分享效率，通过文章管理优化、全文搜索以及AI辅助写作，实现内容创作过程的智能化和便捷化。系统提供从用户注册登录到文章发布编辑的整个内容创作流程支持，确保文章管理的每一环节都尽可能高效、便捷地运行。主要功能有用户认证、文章管理、分类标签、全文搜索、AI辅助、个人中心、文件上传等。
技术栈：Spring Boot 3.2.11 + Spring Security + MyBatis-Plus + Redis + Elasticsearch + MySQL + Vue 3 + Element-Plus + Pinia + MinIO + Docker + Linux + 智普AI
主要职责：
用户认证模块：
用户注册：使用Spring Mail发送邮箱验证码，验证码存储在Redis中并设置5分钟过期时间，后端使用BCrypt加密存储。
用户登录：基于Spring Security集成JWT无状态认证，设计AccessToken（15分钟）+ RefreshToken（7天）双Token机制，RefreshToken存储在Redis中，实现Token自动续期功能。
安全机制：设计Token黑名单机制，使用Redis Set存储被注销的Token，确保用户注销、修改密码等敏感操作后Token立即失效；实现基于Redis的接口限流功能，防止恶意请求。
文章管理模块：
文章发布：使用mavon-editor集成Markdown编辑器，支持文章标题、内容、摘要、封面图上传，支持草稿保存和正式发布两种状态。
文章搜索：集成Elasticsearch实现全文搜索功能，使用IK分词器进行中文分词，支持对文章标题、内容、标签进行全文检索，支持搜索关键词高亮显示，提升搜索性能和用户体验。
分类标签管理：实现文章分类和文章同标签的多对多关联，支持分类自动创建、标签管理和按分类、标签筛选查询。
文章列表：实现文章列表分页查询，支持按创建时间、浏览量、点赞数等多种方式排序。
文章详情：使用Redis缓存文章详情数据，设置10分钟过期时间，减少数据库查询压力；使用Redis实现文章浏览量统计功能。
个人中心模块：
用户信息管理：实现用户昵称、个人简介、密码修改功能
头像上传：集成MinIO对象存储服务，实现用户头像和文章封面的上传功能，提升文件访问性能。
AI辅助功能：
文章润色：集成智普AI（LangChain4j框架），在文章发布时提供内容润色功能，优化文章表达、提升语言流畅度和专业性，保持技术准确性。
文章总结：集成智普AI实现文章摘要自动生成功能，查看文章时自动生成的100字左右的摘要，帮助读者快速了解文章核心内容。
性能优化：
缓存策略：使用Redis String类型缓存文章详情（10分钟过期）和用户信息（30分钟过期），有效降低数据库查询压力。
热门文章：使用Redis ZSet实现热门文章排行榜功能，根据文章浏览量进行排序。
数据统计：使用Redis Set存储点赞用户集合，提高点赞功能查询效率，避免重复点赞。
定时任务：
缓存清理：使用@Scheduled注解实现定时任务，每天凌晨2点自动清理Redis中过期的缓存数据，释放Redis内存空间，避免内存泄漏。
热门文章排行：使用定时任务每天凌晨1点自动统计文章浏览量，生成热门文章排行榜并存储到Redis ZSet中，定时更新排行榜数据，提升用户浏览体验。
部署上线： 在Linux服务器上部署项目，使用Docker容器化部署MySQL、Redis等中间件，熟悉Linux常用命令和Docker操作，完成从开发环境到生产环境的部署工作。
